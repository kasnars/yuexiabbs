# 月下论坛BBS

### 所用技术栈  
前端： Typescript, vue3  
后端： Node.js，koa2，Mongodb, Mongosee 
部署： 腾讯云， centos， PM2

### 线上地址  
[月下bbs](http://kasnars.club/)  
[后端部分GIT仓库](https://gitee.com/kasnars/nodekoa2)  

### 具体功能实现  
- 实现登录注册流程，并用Token进行权限校验  
- 实现文章和专栏的发布评论编辑删除流程  
- 实现图片上传功能和分页等功能  
- 利用vue-router实现前端路由切换等操作  
- 利用vuex + localStorage 进行全局状态管理  
- 利用Bootstrap完成前端样式的快速搭建
